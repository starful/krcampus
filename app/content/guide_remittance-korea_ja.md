# 韓国への海外送金ガイド：Wise（ワイズ）と銀行送金を徹底比較

韓国への留学やワーキングホリデーは、人生において非常にエキサイティングな挑戦です。ソウルの歴史的な宮殿を散策したり、弘大（ホンデ）のトレンディなカフェを巡ったり、本場の韓国グルメを楽しんだりと、楽しみは尽きません。しかし、お気に入りのスーツケースに荷物を詰め込み、D-2（留学）ビザやH-1（ワーホリ）ビザを手にする前に、誰もが直面する現実的なハードルがあります。それが「学費の支払い」と「現地生活費の準備」です。

日本の多くの留学生にとって、これは数万円から数十万円、あるいは数百万円という大金を、韓国の銀行口座（ハナ銀行、ウリ銀行、新韓銀行など）へと国境を越えて送金することを意味します。従来、海外への電信送金といえば、銀行の窓口に出向いて複雑な書類を記入し、高額で不透明な手数料を支払うのが当たり前でした。しかし現在では、フィンテックの進化により「Wise（ワイズ）」などの画期的なオンライン送金サービスが登場し、その常識は180度覆されました。

本記事では、2026年最新の「KR Campus」ガイドとして、Wiseと従来の日本の銀行送金を徹底比較します。手数料、為替レート、送金日数、安全性などを分かりやすく比較し、あなたの大切な資金を最も安く、早く、安全に韓国へ送る方法をご紹介します。

## 1. 韓国送金の壁：高額な初期費用とスケジュール

韓国で学生生活をスタートさせるにあたり、渡航前および到着直後にいくつかの大きな支払いを行う必要があります。代表的な支出は以下の通りです。

* **大学・語学堂の学費:** 1学期あたり約30万円〜80万円程度を一括で支払う必要があります。
* **住居の保証金（ポジュングム / 보증금）:** 韓国でワンルームやオフィステルを契約する際、まとまった保証金（約20万円〜100万円以上）が必要です。この保証金は退去時に全額返金されますが、契約時に一括で送金しなければなりません。
* **毎月の生活費:** 食費、交通費（T-money）、教材費、交際費などを含め、毎月およそ10万〜15万円の予算を見ておく必要があります。

特に大学への学費支払いは、期限を過ぎると「入学許可書（CoA）」の発行が遅れ、最悪の場合、ビザ申請が間に合わなくなる恐れがあります。そのため、迅速かつ確実に届き、金額の誤差が生じない信頼性の高い送金手段が不可欠なのです。

## 2. 従来の銀行送金（SWIFT送金）：昔ながらの手法と隠れたリスク

日本の大手メガバンク（三菱UFJ銀行、三井住友銀行、みずほ銀行）やゆうちょ銀行などは、海外送金のインフラとして世界的なネットワークである「SWIFT（スウィフト）」を利用しています。あなたが日本の銀行から韓国の銀行口座にお金を送る際、資金はいくつかの「仲介銀行（コルレス銀行）」を経由して目的地に到着します。

このSWIFT送金には、主に以下のような「コスト（費用）」が発生します。

### SWIFT送金で発生する4つのコスト：
1. **送金手数料（日本の銀行側）:** 日本の銀行が送金手続きを行うために徴収する手数料です。窓口利用かインターネットバンキング利用かで異なりますが、約2,000円〜6,000円程度かかります。
2. **中継銀行手数料（リフティングチャージなど）:** 資金が韓国に届くまでに経由する複数の海外銀行が差し引く手数料です。送金時に「送金人負担」か「受取人負担」かを選択しますが、いずれにしても合計で2,000円〜4,000円程度が削られます。
3. **被仕向送金手数料（韓国の銀行側）:** 韓国の銀行が海外からの送金を受け取る際にかかる手数料です。通常、10,000ウォン〜20,000ウォン（約1,100円〜2,200円）が受け取り口座の残高から差し引かれます。
4. **為替マークアップ（隠れたコスト）:** 銀行が独自に設定した外貨両替レート（TTSレート）が適用されます。市場の実際の両替レートに対して、通常1%〜3%以上の上乗せ（スプレッド）が含まれており、高額な送金になればなるほど、この「見えない手数料」だけで数万円単位の損をしてしまいます。

## 3. Wise（旧TransferWise）：透明で現代的な選択肢

これに対し、現代の留学生から圧倒的な支持を集めているのが「Wise（ワイズ）」です。WiseはSWIFTネットワークを直接利用せず、世界各国に持っている自社の現地口座同士をマッチングさせる独自のピア・ツー・ピア（P2P）システムを採用しています。

例えば、あなたが日本から韓国へ「日本円（JPY）を韓国ウォン（KRW）」に両替して送金したいとします。このとき、あなたの資金はWiseの日本の銀行口座（三菱UFJ銀行など）に入金され、同時にWiseの韓国にある現地口座から同等額のウォンが韓国の受取口座に振り込まれるという仕組みになっています。資金が国境を越えて実際に移動しないため、コストを大幅に削減できるのです。

### 留学生がWiseを選ぶべき理由：
* **本当の為替レート（ミッドマーケットレート）:** GoogleやYahoo!ファイナンスで検索したときに表示される、上乗せなしの「本物の為替レート」をそのまま適用します。為替レートに手数料が隠されていることは一切ありません。
* **安くて透明な手数料:** 送金手続き時に、日本円でいくらの手数料が発生するかがリアルタイムで1円単位まで表示されます。手数料は送金額の約0.6%〜1.0%程度と極めて格安です。
* **圧倒的なスピード:** 現地国内での振り込み処理となるため、送金手続き完了から数分〜数時間、遅くとも24時間以内には韓国の口座に着金します。
* **スマホひとつで完結:** アプリのUI（操作画面）が非常にシンプルで分かりやすく、日本のマイナンバーカードなどを使ってオンラインで簡単に本人確認が行えます。銀行の窓口で何時間も並ぶ必要はありません。

## 4. 直接対決：Wise vs 伝統的な銀行送金

ここからは、両者の違いをより具体的に理解するために、わかりやすい比較表と具体的なシミュレーションで検証していきます。

### 【表1】機能・コスト比較一覧

| 項目 | 従来の銀行送金（SWIFT） | Wise（ワイズ） |
| :--- | :--- | :--- |
| **為替レート** | 各銀行独自のスプレッド付きレート（1〜3%の上乗せ） | ミッドマーケットレート（上乗せ・隠れコストなし） |
| **日本側の送金手数料** | 高額な固定手数料（約2,000円〜6,000円） | 割安な変動手数料（送金額の約0.5%〜1%） |
| **中継銀行手数料** | 発生する（予測が困難で約2,000〜4,000円） | なし（直接現地送金のため） |
| **韓国側の受取手数料** | 発生する（約10,000〜20,000 KRW） | なし（韓国内の国内振込として着金するため） |
| **着金までの時間** | 3〜5営業日（土日祝を除く） | 数分〜24時間以内（即時着金することも多数） |
| **送金の追跡** | 不可（途中で資金がどこにあるか分からない） | 完全可（アプリ上で進捗をリアルタイムに確認） |

### 【表2】50万円を送金した場合のシミュレーション（実質コスト比較）

それでは、実際に日本の留学生が「50万円」を韓国の大学口座や生活費用として送金する場合をシミュレーションしてみましょう。※為替レートは分かりやすく「1円 = 9.00ウォン（KRW）」のリアルレートを基準とします。

| コスト項目 | 伝統的な銀行送金（SWIFT） | Wise（ワイズ）での送金 |
| :--- | :--- | :--- |
| **元資金（日本円）** | 500,000円 | 500,000円 |
| **送金手数料** | 3,500円 | 3,200円（※送金額に応じた手数料） |
| **適用される為替レート**| 1円 = 8.73ウォン（3%の上乗せスプレッド）| 1円 = 9.00ウォン（市場の実際レート） |
| **中継銀行手数料** | 2,500円（送金時に差し引かれる） | 0円 |
| **受取銀行手数料** | 10,000ウォン（約1,100円相当） | 0円 |
| **最終的な実質送金額** | 494,000円分を8.73で両替し、さらに韓国での受取手数料を引く | 496,800円分を9.00のレートで直接両替 |
| **韓国で受け取る金額**| **4,302,620ウォン** | **4,471,200ウォン** |
| **差額（損失）** | **-168,580ウォン（約18,730円の赤字）** | **0円（最大値で受取）** |

このシミュレーションから明らかなように、同じ「50万円」を日本から送っているにもかかわらず、銀行送金を選ぶだけで**約168,580ウォン（約18,700円分）**も損をしてしまうことになります。1万8,000円もあれば、韓国で美味しいサムギョプサルを何度も楽しんだり、一ヶ月分の交通費をまかなったり、語学堂の新しい教科書を何冊も購入したりできます。留学生にとって、この差額は決して無視できるものではありません。

## 5. 韓国へ送金する前に知っておくべき法制度と注意点

韓国は「外国為替取引法」という法律に基づき、資金の流入・流出を厳格に管理しています。トラブルを防ぐために、以下の点に必ず注意してください。

* **実名確認と外国人登録証（在留カード）の壁:** 韓国で個人の銀行口座を開設するには、入国後に発行される「外国人登録証（外国人登録番号）」が必要です。これの取得には入国後数週間から1ヶ月程度かかるため、渡航直後の最初の学費や家賃は、日本の家族から自分の韓国口座に直接送ることはできません。そのため、最初は大学が用意した学校公式の銀行口座、または大家さんの口座に直接送金することになります。
* **年間5万ドルの壁:** 韓国の規制により、年間累計で50,000米ドル（約750万円相当）を超える海外送金を受け取る場合、韓国の税務当局（国税庁）へ資金使途などの証明書類を提出し、申告を行う義務が発生します。留学費用であれば問題なく承認されますが、書類の準備に時間がかかる点に注意してください。
* **仮想口座（가상계좌：カサンケジャ）への対応:** 韓国の大学や語学堂は、学生一人ひとりに個別の「仮想口座（学費納付専用の一時的な口座）」を割り当てることが一般的です。Wiseはこのような韓国の仮想口座への送金にも対応していますが、送金画面で入力する「受取人名（英語表記）」や「銀行コード」に1文字でも間違いがあると、エラーになり送金が差し戻されてしまいます。学校の納付書（インボイス）をよく確認し、一字一句間違いのないように正確に入力しましょう。

## 6. その他の便利な海外送金サービス

Wiseのほかにも、外国人留学生に人気のある専門的な送金サービスがいくつかあります。それぞれの特徴を簡単に把握しておきましょう。

* **SentBe（セントビー）:** 韓国発のフィンテック企業が運営する海外送金サービスです。特にアジア圏内の送金ルートにおいて非常に強い競争力を持っており、手数料もWiseと同等、あるいはキャンペーン時にはさらに安くなるケースがあります。韓国に特化したサービスであるため、現地でのサポート体制が整っているのが強みです。
* **Flywire（フライワイヤー）:** 世界中の多くの大学や語学堂が、海外からの学費支払い決済プラットフォームとしてFlywireと公式に提携しています。学校のポータルサイトから直接クレジットカード支払いや国内銀行振込のように支払えるため、とにかく「確実で安全、手続きが楽」というメリットがあります。ただし、適用される為替レートはWiseと比べるとやや高めに設定されていることが多いため、利便性とコストのバランスを考慮して選択しましょう。

## 7. KR Campusが推奨する「最も賢い韓国送金ロードマップ」

韓国留学中のコストを最小限に抑えつつ、スムーズに生活をスタートするために、以下のステップで送金を組み立てることをおすすめします。

### 【ステップ1：渡航前】
大学から学費の納付書が届いたら、まずは**Wise**を使って、学校が指定する仮想口座へ直接学費を支払いましょう。Wiseなら送金額の過不足なく、指定されたウォン建ての金額を即座に確実に着金させることができるため、入学許可書がスピーディーに手に入ります。

### 【ステップ2：渡航直後】
韓国に到着してから外国人登録証が発行されるまでの間（約1ヶ月）は、現地での急な出費に対応できるよう、日本の銀行口座からお金を引き出せる海外デビットカードや、**Wiseデビットカード**にお金をチャージして持参しましょう。地下鉄やバスで使う交通カード（T-Moneyカード）へのチャージや、毎日の食事代などの支払いに大変便利です。

### 【ステップ3：外国人登録証の発行後】
外国人登録証が手に入ったら、すぐに韓国の現地銀行（ハナ銀行やウリ銀行など、学校内にある店舗がおすすめ）へ行き、自分の名義の韓国の銀行口座を開設します。また、韓国の便利なモバイル決済（TossやKakao Pay）やオンラインショッピングサイト、デリバリーアプリをフル活用できるように、チェックカード（デビットカード）も同時に発行してもらいましょう。

### 【ステップ4：滞在中の仕送り・生活費】
韓国の自分の口座が開設できたら、日本の家族からの仕送りや生活費の補充は、すべて**Wise**を経由して「日本の自分の口座 ➔ 韓国の自分の口座」へと送金します。これにより、毎月の生活費の両替手数料を数千円単位で継続して節約し続けることができます。

## 8. 韓国送金に関するよくある質問（FAQ）

### Q1. Wiseは本当に安全なサービスですか？お金が消えたりしませんか？
**A. 極めて安全なサービスです。**
Wiseは日本においては関東財務局に「資金移動業者（登録番号：関東財務局長第00040号）」として登録されています。また、顧客から預かった資金は「履行保証金」として、法務局に供託されるほか、国内の主要銀行（三菱UFJ法務信託銀行など）に保全されているため、万が一Wiseが破綻した場合でも送金資金は全額保護されます。さらに世界中で数百万人以上に利用されており、各国の金融当局の規制下で厳重に運営されています。

### Q2. 銀行窓口からの送金でなければ、大学側が海外からの送金として認めてくれないことはありますか？
**A. 基本的にその心配はありません。**
Wiseからの送金であっても、学校の仮想口座に着金した時点で自動的に「〇〇（あなたの名前）からの支払い」としてシステム上で照合されるため、通常通りの納付完了となります。ただし、念のため送金が完了した際にアプリからダウンロードできる「送金証明書（PDF）」を保存しておき、学校から照会があった場合にすぐ提出できるようにしておくと安心です。

### Q3. 送金手続きをしてから、韓国の口座に届くまでどのくらいの時間がかかりますか？
**A. Wiseであれば早ければ数分、遅くとも翌営業日には届きます。**
日本の銀行からWiseの日本口座への振り込み（振込手数料無料枠などを使うとよりお得）が確認され次第、すぐに韓国口座へ着金手続きが行われます。平日の午前中に手続きをすれば、同日午後には韓国の口座にウォンで届くことがほとんどです。一方、日本の伝統的な銀行によるSWIFT送金の場合は、3〜5営業日ほどかかるのが一般的です。

### Q4. 日本のクレジットカードで学校の学費を支払うことはできますか？
**A. 大学によって異なりますが、対応していない、または高い決済手数料がかかる場合があります。**
韓国の多くの大学は学費のクレジットカード決済に対応していません。対応している場合でも、海外発行カードに対して2%〜4%程度の高額なカード決済手数料が上乗せされることが多いため、結果的にWiseで銀行口座へ直接送金する方がはるかに安く抑えられます。

## まとめ

日本の一般銀行を使った従来の海外送金は、手数料や為替の隠れたコストが重なり、大切なお金を大きく削られてしまいます。一方で、Wise（ワイズ）をはじめとする最先端のオンライン送金プラットフォームを活用すれば、無駄なコストを徹底的にカットし、驚くほどスピーディーに、そして安全に韓国へ送金が可能です。

かしこくお金を管理して、余計な手数料を賢く節約しましょう。節約できたお金で、韓国での充実したキャンパスライフを存分に楽しんでください！
