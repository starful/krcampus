# ヨンサン大学校（YSU）留学生ガイド：釜山・海雲台で観光、Webtoon、グローバルビジネスを学ぶ

韓国への留学を検討する際、どの都市で、何を学ぶかは将来のキャリアを左右する極めて重要な決断です。「質の高い実践的な教育」「充実したインターンシップ」「刺激的で快適なライフスタイル」をすべて叶えたい学生にとって、韓国第2の都市・釜山（プサン）に位置する**ヨンサン大学校（Youngsan University / 霊山大学校）**は、非常に魅力的な選択肢です。

美しいビーチリゾートとして世界的に有名な「海雲台（ヘウンデ）」にキャンパスを構え、観光、ホテル、ウェブトゥーン、アニメーション、スマート産業など、時代の最先端を行く実務教育を提供しているヨンサン大学校。本ガイドでは、大学の概要から学部、学費、奨学金、そして釜山でのキャンパスライフまで、留学生に必要な情報を網羅して解説します。

---

## 1. ヨンサン大学校の概要

ヨンサン大学校（영산대학교）は、産業界のニーズに直結した「実務型専門人材」の育成に特化した私立大学です。同校は、韓国の産業構造に合わせた2つのキャンパス（釜山・海雲台キャンパス、慶尚南道・梁山キャンパス）を運営しています。留学生の多くが在籍するメイン拠点となるのが、韓国トップクラスの国際観光都市である釜山広域市に位置する**海雲台（ヘウンデ）キャンパス**（住所：*釜山広域市海雲台区盤松路142*）です。

留学生の**合格率は約80%**と門戸が広く開かれており、基礎的な学習意欲と語学要件を満たしていれば入学のチャンスが十分にあります。現在、世界中から**1,250人以上の外国人留学生**が集まって学んでおり、キャンパス内は非常にグローバルで活気にあふれた雰囲気に包まれています。地元釜山の観光・IT・デジタルコンテンツ産業と緊密に連携したカリキュラムを導入しており、卒業後の高い就職率が最大の強みです。

## 2. 留学生向けの語学プログラムとサポート体制

ヨンサン大学校では、韓国語に不安がある学生や、英語での学位取得を目指す学生のために、多様な言語トラックとサポート体制を用意しています。

*   **グローバルビジネス＆観光トラック:** 「グローバルビジネス大学」や「ホテル観光大学」の一部の講義では、英語での講義や、英語の教材・資料を用いたサポートが行われています。
*   **韓国語研修課程（語学堂）:** 大学付属の言語教育院では、1クラス約15人の少人数制で、年4学期（春・夏・秋・冬）の韓国語集中プログラム（1期10週間）を提供しています。初心者から始めて、最短1年で本科（学部）入学に必要なTOPIK（韓国語能力試験）3級・4級以上の取得を目指すことができます。
*   **1対1のメンター制度（バディ制度）:** 入学直後の外国人留学生に対して、韓国人学生や先輩留学生が「メンター」としてペアになり、大学の履修登録から釜山での日常生活のセットアップ（外国人登録、携帯電話の契約、銀行口座開設など）までを親身にサポートします。

### 代表的な看板学科のクローズアップ：

#### ① ホテル観光・調理芸術学科（海雲台キャンパス）
釜山・海雲台は、グランド朝鮮、シグニエル、パラダイスホテル、パークハイアットなど、最高級の5つ星グローバルホテルが密集する韓国最大の観光リゾート地です。この立地を活かし、学生たちは在学中から一流ホテルでの実務実習や、国際コンベンションセンター（BEXCO）でのMICEイベントの運営に深く関わることができます。世界的な料理コンテストでの受賞歴を誇る教授陣からの直接指導も大きな魅力です。

#### ② ウェブトゥーン（Webtoon）＆アニメーション学科
日本でも「LINEマンガ」や「ピッコマ」を通じて大ヒットしている縦スクロール漫画「Webtoon」。ヨンサン大学校のウェブトゥーン学科は、韓国国内でもトップクラスの実績を誇り、現役で活躍するプロのウェブトゥーン作家やイラストレーターが教鞭を執っています。大学にはプロと同じ仕様の液晶ペンタブレット（Wacom Cintiq等）がずらりと並ぶ専用スタジオが用意され、学生は在学中から作品をプラットフォームに連載したり、アシスタントとしての実務経験を積んだりすることができます。

## 4. 学費・生活費と留学生向け奨学金制度

ヨンサン大学校は、ソウル首都圏の私立大学と比較して学費や生活費が手頃であり、さらに留学生向けの非常に手厚い奨学金制度を整えているため、経済的な負担を最小限に抑えて留学することができます。

### 留学生の年間想定費用（2024/2025学年度基準）

| 費用項目 | 韓国ウォン（KRW） | 日本円（参考・目安）* |
| :--- | :--- | :--- |
| **入学金（初回のみ）** | 250,000 KRW | 約28,000円 |
| **年間授業料（平均）** | 6,800,000 KRW | 約750,000円 |
| **学生寮費（1学期/約4ヶ月）** | 1,200,000 KRW | 約132,000円 |
| **国民健康保険料（月額）** | 75,000 KRW | 約8,300円 |

*※レートは 100 KRW ≒ 11円 として算出。学費は学部（文系・理系・芸術系）によって多少異なります。*

### TOPIK（韓国語能力試験）に基づく優秀奨学金（学部新入生対象）

ヨンサン大学校では、入学時の韓国語能力に応じて、最初の学期の授業料が大幅に免除される制度を設けています。また、入学後も一定の成績（GPA）を維持することで、卒業まで奨学金を受給し続けることが可能です。

| 入学時の保有TOPIK級 | 授業料減免率 | 継続要件 / 条件 |
| :--- | :--- | :--- |
| **TOPIK 5級 または 6級** | **100% 全額免除** | 入学後、前学期のGPA 3.5以上を維持 |
| **TOPIK 4級** | **50% 半額免除** | 入学後、前学期のGPA 3.0以上を維持 |
| **TOPIK 3級** | **40% 免除** | 入学後、前学期のGPA 2.5以上を維持 |
| **TOPIK 2級 / 語学堂修了者** | **30% 免除** | 実技系学科または指定学科に適用。GPA 2.5以上 |

このように、日本でしっかりと韓国語を準備してTOPIK 4級〜5級を取得しておけば、最初の学期は日本の国公立大学や私立大学に通うよりもはるかに安い学費で、質の高い教育を受けることが可能です。

## 5. 外国人留学生の入学選考プロセス

日本からの志願者は、書類選考および面接（オンライン可）を通じて選考されます。学業成績だけでなく、学びへの情熱や明確な将来設計が評価されます。

### 主な出願要件：
1.  **学歴要件:** 本科（学士課程）への入学は、国内外で12年以上の学校教育を修了（高校卒業または卒業見込み）していること。大学院への入学は学士号を保持していること。
2.  **語学要件:**
    *   *韓国語トラック:* TOPIK 3級以上（芸術・体育系の一部の学科はTOPIK 2級以上でも出願可能な場合があります）。
    *   *英語トラック:* TOEFL iBT 71、IELTS 5.5、またはそれに相当する公認英語資格。
3.  **財政能力証明:** 韓国の学生ビザ（D-2）の申請に必要な、申請者本人または経費支弁者の名義の預金残高証明書（目安として16,000〜18,000米ドル相当、銀行に約1ヶ月以上凍結保存されたもの）。

### 出願から入学までの4ステップ：
*   **ステップ 1（オンライン出願）:** 大学の公式出願ポータルから願書を提出し、高校の卒業証明書や成績証明書などの必要書類（アポスティーユまたは領事認証が必要）を郵送します。
*   **ステップ 2（選考料の支払い）:** 出願手数料を支払い、書類審査を受けます。
*   **ステップ 3（面接試験）:** オンライン（Zoomなど）で個別面接を実施します。主に志望動機、言語能力、学習計画が問われます。
*   **ステップ 4（合格発表とビザ申請）:** 合格後に学費を納入すると、大学から「標準入学許可書（CoA）」が発行されます。これを用いて、日本国内の韓国領事館・大使館で留学ビザ（D-2）を申請します。

## 6. キャンパスライフと釜山・海雲台での暮らし

ヨンサン大学校での学生生活は、勉強だけでなく、韓国で最も美しく活気のある街の一つである「釜山」を存分に楽しむことができる素晴らしい環境が整っています。

*   **最高のロケーション:** 海雲台キャンパスは、釜山地下鉄4号線の駅と直接結ばれており、交通アクセスが抜群です。放課後や週末には、地下鉄やバスを利用してすぐに有名な海雲台ビーチ、カフェ通り、伝統市場、アジア最大級のショッピングモール「新世界百貨店センタムシティ店」などに行くことができます。
*   **快適な学生寮（ドミトリー）:** 慣れない海外生活をサポートするため、外国人新入生には優先的に学生寮の部屋が割り当てられます。寮内はWi-Fi、エアコン、個別暖房が完備されており、2人部屋または4人部屋を選択できます。共同スペースには、フィットネスジム、スタディルーム、洗濯室、コンビニエンスストア、カフェがあり、安全で快適な生活が約束されています。
*   **活発なサークル活動とイベント:** K-POPダンス、バンド、サッカー、テコンドー、語学交換など、多種多様なサークルが存在し、韓国人学生と日常的に交流することができます。また、毎年開催される学園祭では、有名アーティストのライブステージや各国の留学生が自国の伝統料理を振る舞うブースが出店され、キャンパス中が大いに盛り上がります。

## 7. よくある質問（FAQ）

### Q1. 日本人留学生にとって、ヨンサン大学校の合格難易度はどれくらいですか？
合格率は約80%となっており、基本的な高校の成績と語学要件（TOPIK 3級以上、または英語トラックのスコア）をクリアしていれば、合格の可能性は非常に高いです。面接では「なぜこの学科で学びたいのか」「卒業後にどうなりたいか」という熱意を伝えることが重要視されます。

### Q2. 留学中にアルバイトをすることはできますか？
はい、可能です。留学生ビザ（D-2）を保有している学生は、入学後1学期（約6ヶ月）を修了し、一定の出席率と成績（GPA）を満たしていれば、出入国在留管理庁の許可を得て週20時間まで（長期休暇中は制限なし）アルバイトをすることができます。観光地である海雲台エリアには、カフェ、レストラン、ホテル、免税店など、日本語や英語、韓国語を活かせるアルバイト求人が数多くあります。

### Q3. 韓国語が全く話せなくても入学できますか？
語学力がない場合は、まずヨンサン大学校の「韓国語研修課程（語学堂）」に入学することをお勧めします。そこで最短半年〜1年間集中的に韓国語を学んでTOPIKを取得した後、学部の希望する学科へスムーズに内部進学（新入学または編入学）することができます。また、グローバルビジネス分野では、英語を中心に学ぶトラックも用意されています。

### Q4. 釜山での生活費は、ソウルと比べてどれくらい安いですか？
釜山はソウルに比べて家賃や食費が20%〜30%ほど安く抑えられる傾向があります。特に学生寮の費用や大学周辺の飲食店は学生向けにリーズナブルに設定されており、毎月の生活費（食費、交通費、お小遣い含む）は5万〜8万円程度で十分に快適な暮らしを送ることができます。

### Q5. 卒業後の進路（就職）はどうなっていますか？
ヨンサン大学校は実務教育に強みがあるため、韓国国内のグローバル企業、ホテル業界、IT企業、ウェブトゥーン制作スタジオなどに多くの卒業生を輩出しています。また、日本に帰国して、韓国語や観光・ITの専門知識を活かして日系企業や外資系企業に就職する道を選ぶ学生も多く、大学の就職支援センター（Career Center）が個別で履歴書の添削や模擬面接のサポートを行っています。
