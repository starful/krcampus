# 韓国政府奨学金（GKS/KGSP）完全ガイド

韓国への留学は、世界中の何百万人もの学生にとっての夢です。世界トップクラスの教育システム、最先端の技術、そして世界的な文化の波（韓流）で知られる韓国は、留学生に大人気の目的地となっています。しかし、海外での教育資金を確保することは簡単なことではありません。

そこで登場するのが、かつてKGSP（Korean Government Scholarship Program）として知られていた**大韓民国政府奨学金（GKS）**です。国立国際教育院（NIIED）が運営するGKSは、世界で最も手厚く、権威のある全額給付型奨学金の一つです。授業料全額、往復航空券、毎月の生活費、医療保険、さらには1年間の集中韓国語研修費用までカバーされます。

申請を計画している方に向けて、この完全ガイドでは、募集枠（トラック）、応募資格、必要書類の準備、スケジュール、そして選考を勝ち抜くためのコツまで、詳しく解説します。

## 1. GKSの募集枠を理解する：大使館推薦 vs 大学推薦

GKSに応募する際、まず最初に重要な選択をしなければなりません。それは、**大使館推薦（Embassy Track）**または**大学推薦（University Track）**のどちらで申請するかです。同じ募集回に両方へ応募することはできず、重複申請した場合は即座に失格となります。

これら2つの募集枠の違いを理解することは、申請戦略を立てる上で非常に重要です。

### 主な違いの比較

| 項目 | 大使館推薦 | 大学推薦 |
| :--- | :--- | :--- |
| **申請先** | 居住国にある韓国大使館・領事館 | 志望する韓国の大学へ直接 |
| **志望大学の選択肢** | 3つの大学を選択する必要あり（Type AおよびBのリストから） | 1つの大学のみ選択可能 |
| **選考ラウンド** | 3次選考（大使館 $\rightarrow$ NIIED $\rightarrow$ 各大学） | 2次選考（大学 $\rightarrow$ NIIED） |
| **選考枠・定員** | 国別の割り当て枠 | 大学別・専攻別の割り当て枠 |
| **柔軟性** | 高い（1つの大学に落ちても、他の大学に合格する可能性がある） | 低い（一発勝負） |

### 必須の韓国語研修プログラムと学位課程

GKSのユニークな特徴の一つに、**1年間の必須韓国語研修プログラム**があります。

学位課程の授業が英語で行われるか韓国語で行われるかに関わらず、すべてのGKS奨学生は、最初の1年間を指定された語学教育機関で過ごし、少なくとも**TOPIK 3級以上**を取得する必要があります…
