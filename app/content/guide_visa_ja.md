# 韓国留学ビザ完全ガイド：D-2（正規留学）とD-4（語学留学）の申請手順と注意点

韓国へようこそ！世界最高水準の教育環境と、魅力あふれるK-POP、ファッション、グルメなどの豊かなカルチャーが融合した韓国での留学生活は、素晴らしい経験となるはずです。しかし、ソウルや地方都市でのキャンパスライフをスタートさせる前に、必ずクリアしなければならない重要なステップがあります。それが「学生ビザ」の取得です。

韓国の学生ビザには、大学や大学院への進学を目指す方のための「D-2ビザ（留学）」と、大学付属の語学堂などで韓国語を学ぶ方のための「D-4ビザ（一般研修）」の2種類があります。このガイドでは、KR Campus（韓国留学）の専門編集部が、それぞれのビザの特徴、申請に必要な書類、手続きの流れ、そして韓国入国後に必要な手続きまで分かりやすく徹底的に解説します。これさえ読めば、複雑なビザ申請もスムーズに進められます！

---

## 1. D-2ビザとD-4ビザの違いを理解する（どちらを選ぶべき？）

韓国の学生ビザを申請するにあたって、まず最初に行うべきことは、自身の留学目的に適した正しいビザの種類を選択することです。学位取得を目指すのか、語学力の向上を目指すのかによって、申請するビザが大きく異なります。

### D-2ビザ（正規留学ビザ）

D-2ビザは、専門大学、大学（学士課程）、大学院（修士・博士課程）などの高等教育機関において、正規の学位課程を履修する学生、または交換留学生を対象としたビザです。
- **特徴**: 滞在可能期間が比較的長く（通常1〜2年単位）、学業を継続する間は現地での在留延長がスムーズです。また、一定の条件を満たせば、韓国到着後比較的早い段階でアルバイトを始めることができます。

### D-4ビザ（一般研修・語学留学ビザ）

D-4ビザは、大学付属の韓国語教育機関（語学堂）などで、学位取得を目的としない語学研修を行う学生を対象としたビザです。
- **特徴**: 一般的に「語学留学」と呼ばれる場合の多くはこのビザに該当します。最初の有効期間は通常6ヶ月で、出席率や成績に応じて最大2年間まで韓国現地で延長手続きを行うことができます。

D-2ビザとD-4ビザの主な違いを以下の表にまとめました。

| 項目 | D-2（正規留学・学位課程ビザ） | D-4（一般研修・語学留学ビザ） |
| :--- | :--- | :--- |
| **主な目的** | 学位取得（学士、修士、博士、専門学士）、交換留学 | 語学堂などでの韓国語研修、非学位課程の受講 |
| **受入教育機関** | 韓国国内の認可された大学・大学院・専門大学 | 大学付属 of 語学堂、語学研修機関 |
| **初回の滞在期間** | 通常1年間、または2年間（課程による） | 通常6ヶ月間（受講する学期数に基づく） |
| **現地でのアルバイト** | 入学から1学期（通常約6ヶ月）経過後に許可可能 | 入国後6ヶ月経過後、出席率90%以上などの条件で許可可能 |
| **必要な語学要件（TOPIK）** | 原則としてTOPIK 3級以上（英語トラックはTOEFLなど） | 事前の語学能力要件なし（初心者でも申請可能） |
| **延長時の要件** | 一定以上のGPA（学期成績）の維持 | 授業への出席率90%以上が必須要件 |

※注意：どちらのビザを保持している場合でも、留学生としての「本分（学業）」を全うすることが厳格に求められます。特にD-4ビザの場合、出席率が90%を下回ると、次学期のビザ延長が拒否されたり、最悪の場合は強制帰国の対象となることがあるため、日頃の自己管理が極めて重要です。

## 2. ステップ1：大学への出願と「標準入学許可書（COA）」の取得

韓国での学生ビザ取得に向けた第一歩は、現地の学校から「公式な入学許可」を得ることから始まります。韓国政府が認定した教育機関からスポンサーシップ（保証）を得ていない限り、ビザの申請を進めることはできません。出願から入学許可書が手元に届くまでの一般的なプロセスは以下の通りです。

* **希望する大学・語学堂への出願**: 学校の募集要項を確認し、願書を提出します。必要な書類（最終学歴の証明書、成績証明書、自己紹介書、語学能力証明書など）を漏れなく揃えてオンラインまたは郵送で申請します。
* **授業料（学費）の納付**: 出願書類の審査を通過して合格すると、学校から学費の請求書（インボイス）が届きます。指定された期日までに、海外送金や指定の決済プラットフォームを利用して、授業料（通常は最初の1学期または2学期分）を支払います。
* **「標準入学許可書（COA）」の受け取り**: 学校側で学費の着金が確認されると、ビザ申請に不可欠な**「標準入学許可書（Certificate of Admission: COA）」**と、その学校の**「事業者登録証（の写し）」**が発行され、メールまたは郵送で手元に届きます。

**【査証発給認定番号について】**
一部の大学や語学堂（特に韓国法務部が指定する不法滞在率不足優秀大学など）では、学校側が学生の代わりに韓国の出入国管理事務所にビザの事前審査を申請し、**「査証発給認定番号（Visa Issuance Confirmation Number）」**を取得してくれる場合があります。この番号が発行された場合は、物理的な標準入学許可書の代わりに、この認定番号を使って大使館や領事館で簡易的な手続きでビザを申請することができます。

## 3. ステップ2：ビザ申請に必要な書類の準備

入学許可書（または査証発給認定番号）を入手したら、次はいよいよ日本の韓国大使館・領事館、または指定のビザ申請センター（KVAC）に提出する書類一式を準備します。申請者の国籍や管轄の領事館によって多少の違いはありますが、原則として以下の書類が必要となります。

* **査証発給申請書（Visa Application Form）**: 韓国大使館の公式ウェブサイトから最新の書式をダウンロードし、英語または韓国語で正確に記入します。手書きの場合は黒のボールペンを使用し、楷書体でハッキリと記載してください。
* **パスポート（旅券）**: 有効期限が韓国滞在予定期間より「6ヶ月以上」残っている必要があります。また、ビザを貼り付けるための余白ページ（査証欄）が2ページ以上残っていることを確認してください。
* **証明写真**: 申請書に貼付するカラー写真が1枚必要です。サイズは「横3.5cm × 縦4.5cm」で、背景は無地の白、申請日から6ヶ月以内に撮影された、帽子やサングラスを着用していない正面を向いたものに限られます。
* **標準入学許可書（COA）**: 学校から発行された原本、またはPDFを印刷したものを提出します（査証発給認定番号を利用する場合は、その番号が記載された通知書または番号そのものを申請書に記入します）。
* **最終学歴の立証書類（卒業証明書など）**: 高校や大学などの最終学歴の卒業証明書が必要です。日本国内で発行された書類を提出する場合、原則として**「アポスティーユ（Apostille）」**の取得、または日本の駐日韓国大使館（領事館）での領事確認が求められます。アポスティーユは日本の外務省に郵送または窓口で申請することで取得できます。
* **財政能力立証書類（残高証明書）**: 韓国での留学期間中に自活できるだけの十分な資金があることを証明するため、銀行発行の「英文残高証明書」を提出します。必要な基準額の目安は以下の通りです。
  - **D-2ビザ（ソウル首都圏の大学）**: 20,000米ドル相当額以上（日本円で約300万円以上）
  - **D-2ビザ（地方都市の大学）**: 16,000〜18,000米ドル相当額以上（日本円で約240万〜270万円以上）
  - **D-4ビザ（語学留学）**: 10,000米ドル相当額以上（日本円で約150万円以上）
  ※残高証明書は、申請日から30日以内に発行されたものである必要があります。また、本人名義以外の口座（両親の口座など）を使用する場合は、家族関係を証明する書類（戸籍謄本など）と親からの「資金提供誓約書」を併せて提出する必要があります。
* **結核健康診断書（Tuberculosis Test Certificate）**: 韓国政府が指定する「結核高リスク国」の国籍を保持している場合、または日本の特定の管轄領事館において提出が義務付けられている場合に必要となります。日本の一般市民の場合、基本的には免除されることが多いですが、過去に結核高リスク国に一定期間滞在していた履歴がある場合などは提出を求められることがあるため、必ず事前に居住地を管轄する領事館のウェブサイト等で最新情報を確認してください。

## 4. ステップ3：ビザ申請の提出方法とスケジュール（タイムライン）

すべての書類が揃ったら、住民登録のある地域を管轄する駐日大韓民国大使館、領事館、または駐日本大韓民国大使館 領事部（東京の場合はKVAC: 韓国ビザ申請センター）に申請を行います。現在は東京や大阪など一部の地域では、ビザ申請センター（KVAC）を介しての申請が義務付けられている場合がありますので注意してください。

一般的な手続きのタイムラインと必要な費用の目安は以下の通りです。

| プロセスの段階 | 推定される所要期間 | 費用の目安（円換算 / 参考値） |
| :--- | :--- | :--- |
| **大学への出願から合格発表まで** | 入学の2〜3ヶ月前 | 約5,000円〜15,000円（出願料・選考料） |
| **授業料納付後、入学許可書（COA）の発行** | 合格発表後、2〜4週間 | 無料（学費そのものは別途必要） |
| **領事館やKVACへのビザ申請から発給まで** | 申請後、約10〜15営業日（約2〜3週間） | 日本国籍者は査証手数料無料（KVAC利用時は申請代行手数料約2,000円が必要） |
| **韓国入国後の外国人登録証（在留カード）取得** | 入国後、約3〜4週間 | 30,000韓国ウォン（手数料） |

### スムーズな申請のためのアドバイス：
* **余裕を持って準備を開始する**: ビザの申請は、韓国への出発予定日の「少なくとも6〜8週間前」には行うことを強く推奨します。特に春学期（3月入学）に向けた1月〜2月、秋学期（9月入学）に向けた7月〜8月は、世界中からの留学申請が集中するため、審査に通常以上の時間がかかることがあります。
* **居住地の管轄を確認する**: 韓国の領事館は日本国内に複数（東京、大阪、札幌、仙台、横浜、新潟、名古屋、神戸、広島、福岡）あり、住民票のある地域によって申請すべき窓口が厳格に決まっています。異なる管轄の窓口に申請しても受け付けてもらえませんので、事前に必ず自身の管轄領事館を確認してください。
* **すべての書類のコピー（スキャンデータ）を手元に残す**: 大使館に提出した原本書類は、原則として返却されません。韓国到着後に大学の事務手続きや、現地での外国人登録証申請の際に提出を求められるケースが多々あるため、提出前にすべての書類のコピーを取り、スマートフォンなどでPDFデータとして保存しておくことをお勧めします。

## 5. ステップ4：韓国到着後の必須手続きと現地での生活ルール

おめでとうございます！無事にビザが発給され、韓国の土を踏むことができました。しかし、あなたのビザに関する手続きはこれで終わりではありません。韓国内で合法的に、そして安心して留学生活を送るためには、入国後に以下の極めて重要な手続きを行う必要があります。

* **外国人登録証（在留カード：ARC）の申請**: 90日を超えて韓国に滞在するすべての外国人は、入国後90日以内に、自身が居住するエリアを管轄する出入国管理事務所で「外国人登録」を行い、外国人登録証（在留カード）の交付を受ける義務があります。
  - **注意点**: 出入国管理事務所は完全予約制となっており、韓国入国管理の総合ポータルサイト「HiKorea（hikorea.go.kr）」から事前に訪問予約（訪問予約 / Reserve Visit）を行う必要があります。新学期の開始直後は予約が1ヶ月以上先まで埋まってしまうことが多いため、韓国に到着したら、あるいは入国前でも日程が確定した時点で、速やかに予約を入れるようにしてください。
* **韓国国民健康保険（NHI）への加入**: 韓国に滞在する留学生は、外国人登録が完了した時点、または入国から一定期間が経過した時点で、韓国の国民健康保険への加入が義務付けられています。保険料は毎月自動的に請求され、指定された口座やコンビニなどで支払います。加入により、韓国国内の病院で現地の学生と同じように低価格で充実した医療サービスを受けることができます。
* **アルバイト（資格外活動許可）のルールを守る**: 韓国では、学生ビザ（D-2、D-4）を保持しているだけでは、すぐにアルバイトを始めることはできません。不法就労とみなされた場合、罰金刑や強制退去などの厳しい処分が下されます。アルバイトを行うには、必ず事前に「資格外活動許可」を申請し、許可を得る必要があります。
  - **D-2ビザ保持者**: 通常、入学から最初の1学期が終了した後に申請が可能となります。週あたりの勤務時間はTOPIKの級数や成績によって制限されます（一般的に週20〜30時間以内）。
  - **D-4ビザ保持者**: 韓国入国後「6ヶ月（180日）」が経過し、かつ直近の学期の出席率が90%以上である場合に申請資格が得られます。週あたりの勤務時間は週15時間以内などに制限されます。
  ※どちらの場合も、大学の国際交流課（留学生サポート担当）の推薦と、出入国管理事務所の最終的な許可が必要となります。

## 6. よくある質問（FAQ） - 韓国留学ビザに関する疑問を解決

ここでは、KR Campusに寄せられる韓国留学ビザに関する代表的な質問と回答をまとめました。不安な点がある方は、事前にしっかりと確認しておきましょう。

### Q1. 親名義の銀行残高証明書を使ってビザを申請できますか？
**A1.** はい、申請可能です。ただしその場合、資金提供者がご両親であることを証明するために、ご自身とご両親の関係が記載された「戸籍謄本（原本）」、およびご両親が留学費用を全額支援することを誓約した「資金提供誓約書（身元保証書）」を併せて提出する必要があります。日本語の戸籍謄本の場合は翻訳の添付を求められることがあるため、申請する領事館の個別ルールを事前にご確認ください。

### Q2. 語学留学（D-4）から大学進学（D-2）へ、韓国現地でビザを切り替えることはできますか？
**A2.** はい、韓国国内の出入国管理事務所で切り替え手続きを行うことが可能です。語学堂で韓国語を学び、TOPIKなどの必要要件を満たして大学や大学院に合格した場合、日本へ一度帰国することなく、必要書類を揃えて在留資格変更申請を行うことができます。ただし、申請期限や手続きの混雑具合を考慮し、合格が決まったらできるだけ早く国際交流課の担当者に相談し、手続きを進めてください。

### Q3. ビザの有効期限が切れそうな場合、韓国内で延長手続きは可能ですか？
**A3.** はい、可能です。ビザの満了日の約4ヶ月前から満了日当日までに、居住地を管轄する出入国管理事務所（またはHiKoreaでのオンライン申請）にて延長手続きを行うことができます。延長の際には、在学証明書、成績証明書（D-4の場合は出席率証明書）、手数料、滞在地の証明書類（賃貸契約書など）が必要となります。特にD-4ビザの場合、出席率が極めて厳しくチェックされます。

### Q4. 留学中に日本へ一時帰国する場合、特別な手続きは必要ですか？
**A4.** 外国人登録証（在留カード）の交付を受けた後であれば、特別な手続きなしで一時帰国し、再度韓国に入国することができます（再入国許可免除制度が適用されるため）。ただし、外国人登録証の発行前に韓国から出国してしまうと、取得した学生ビザが失効（無効化）してしまうため、必ず外国人登録証が手元に届くまでは韓国国外への出国は避けてください。

### Q5. 銀行の残高証明書は日本語のものでも受け付けられますか？
**A5.** 原則として受け付けられません。ビザ申請書類として提出する残高証明書は、「英語（英文）」または「韓国語」で発行されたものである必要があります。日本のほとんどの都市銀行や地方銀行、ネット銀行（ゆうちょ銀行などを含む）では、窓口やオンライン of マイページから「英文での残高証明書」の発行依頼が可能です。発行までに数日から1週間程度かかる場合もありますので、余裕を持って銀行に申請しておきましょう。
