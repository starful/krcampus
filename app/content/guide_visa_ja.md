# 韓国留学ビザ完全ガイド：D-2とD-4の申請ステップ

KR Campusへようこそ！大好きな韓国で学ぶ日々は、質の高い教育に触れるだけでなく、豊かな文化や刺激的なシティライフを体験できる素晴らしいチャンスです。しかし、ソウルでの新生活を始める前に、必ずクリアしなければならない重要なステップがあります。それが「学生ビザ（査証）」の取得です。

「ビザの手続きが複雑そうで不安…」「何から手を付ければいいのかわからない」と悩んでいませんか？韓国留学を成功させるためには、正確なビザ情報の把握と入念な事前準備が欠かせません。韓国の学生ビザには、主に大学や大学院などの正規課程に進学する人向けの **D-2ビザ** と、大学付属の語学堂などで韓国語を学ぶ人向けの **D-4ビザ** の2種類があります。この記事では、それぞれのビザの特徴、申請に必要な書類、手続きの流れ、そして入国後の注意点まで、分かりやすく徹底解説します。

---

## 1. 留学目的に合わせたビザの選択：D-2ビザ vs D-4ビザ

どちらのビザを申請すべきかは、あなたの留学目的（教育課程）によって決まります。学部、修士、博士課程などの学位取得を目指す場合は「D-2ビザ」が必要です。一方、大学付属の言語教育機関（語学堂）で韓国語のみを学ぶ場合は「D-4ビザ（一般研修）」が該当します。これら2つのビザは、滞在可能期間、アルバイト（資格外活動）の制限、申請要件などが大きく異なります。

なお、一口にD-2ビザと言っても、細かく分類されています。例えば、一般の4年制大学に通う場合は「D-2-2」、大学院修士課程は「D-2-3」、そして日本の在籍大学から派遣される交換留学の場合は「D-2-6」といったサブカテゴリーが存在します。基本の申請プロセスは似ていますが、必要書類が若干異なる場合があるため、自身の留学形態をしっかりと把握しておきましょう。

以下の比較表で、主な違いを確認してみましょう。

| 項目 | D-2（留学ビザ） | D-4（一般研修・語学留学ビザ） |
| :--- | :--- | :--- |
| **主な目的** | 正規学位課程（学士・修士・博士）の履修 | 語学研修（大学付属の語学堂など・非学位） |
| **受入機関** | 韓国の認可大学・大学院 | 大学付属の韓国語教育機関（語学堂） |
| **初回滞在期間** | 通常 1〜2年（課程による） | 通常 6ヶ月（学期ごとの登録・出席率で延長可能） |
| **アルバイト（時間外就業）** | 入国後、最初の1学期が終了してから可能 | 入国後 6ヶ月経過してから可能 |
| **必要な語学力（TOPIK）** | 入学時に一定水準（通常TOPIK 3〜4級以上）が必要 | 事前の語学力は不問（初心者でも申請可能） |

どちらのビザを保持する場合でも、学業に専念し、良好な成績や出席率を維持することが求められます。特にD-4ビザの場合、次学期のビザ延長には「出席率90%以上」が極めて重要な基準となります。出席率が著しく低い場合、ビザの延長が認められず、強制帰国となるケースもあるため十分注意しましょう。また、D-2ビザの学生も、一定以上のGPA（平均評定）を維持しなければビザの延長申請時に追加書類を求められることがあります。

## 2. ステップ1：大学・語学堂への出願と入学許可書（COA）の取得

ビザを申請するためには、まず韓国の教育機関（大学や語学堂）から正式に入学を許可されなければなりません。スポンサーとなる学校がなければ、ビザの申請自体を行うことができません。出願から入学許可書を受け取るまでの一般的な流れは以下の通りです。

* **希望校への出願:** 大学または語学堂のWebサイトから出願手続きを行います。最終学歴の証明書や成績証明書、語学能力証明書（正規留学の場合はTOPIKや英語資格、語学堂の場合は不要なことが多い）を提出します。
* **授業料（学費）の支払い:** 書類審査を通過すると、大学から学費の請求書（Invoice）が届きます。指定された海外口座へ、銀行送金などを利用して速やかに学費（通常は最初の1学期分、語学堂の場合は2学期分以上）を支払います。
* **必要書類の受領:** 支払いが確認されると、学校からビザ申請に必要な**「標準入学許可書（Certificate of Admission: COA）」**と、その学校の「事業者登録証（コピー）」が郵送またはPDFで送られてきます。

韓国の学校制度は日本と異なり、新学期が「3月（春学期）」から始まります。もう一つの主要な入学時期は「9月（秋学期）」です。
一般的に、3月入学のための出願期間は前年の9月〜11月頃、9月入学のための出願期間は同年の3月〜5月頃に設定されています。語学堂の場合は年4回（3月、6月、9月、12月）入学時期があるのが一般的で、出願締め切りは各開講日の約2〜3ヶ月前となっています。ビザの取得が間に合わないという事態を避けるためにも、志望校のスケジュールを早めに確認しておきましょう。

※一部の政府公認優良大学（不法滞在率が低い認証大学）では、紙の標準入学許可書の代わりに、韓国の法務部（出入国管理局）が発行する「査証発給認定番号（Visa Issuance Confirmation Number）」が発給される場合があります。この番号があれば、大使館でのビザ発給手続きが非常にスムーズになり、必要書類も大幅に簡略化されます。

## 3. ステップ2：ビザ申請に必要な書類の準備

標準入学許可書（COA）または査証発給認定番号を手に入れたら、いよいよ駐日韓国大使館や領事館に提出するビザ申請書類一式を準備します。申請者の国籍や管轄の領事館によって多少異なる場合がありますが、日本国籍の方が日本国内で申請する場合の一般的な必要書類は以下の通りです。

* **査証発給申請書:** 大使館のホームページからダウンロードし、必要事項を記入します（英語または韓国語で記入）。証明写真を指定の位置に貼付します。
* **パスポート（原本およびコピー）:** 残存有効期間が予定滞在期間＋6ヶ月以上残っており、査証欄に余白があるもの。
* **証明写真（1枚）:** 3.5cm × 4.5cm、背景は白、最近6ヶ月以内に撮影されたカラー写真。
* **標準入学許可書（COA）:** 大学から送られてきた原本、またはPDFを印刷したもの。
* **最終学歴証明書（卒業証明書など）:** 最終学校の卒業証明書。原則として、日本国外務省の発行する「アポスティーユ（Apostille）」の取得が必要になるケースが多いため、事前に準備しておきましょう。
* **残高証明書（資金証明）:** 留学生活を維持できる資金があることを証明する、銀行発行の英文残高証明書。目安として、ソウル首都圏のD-2ビザ申請には約20,000米ドル以上、地方大学の場合は約16,000〜18,000米ドル以上、D-4ビザ（語学留学）の申請には約10,000米ドル以上の残高が必要です（※日本円での換算額でも可）。発行から30日以内のものを用意します。
* **結核健診診断書:** 特定の結核高リスク国に該当する国籍の方、または過去の渡航歴によって提出を求められる場合があります。日本国籍の方は原則不要とされることが多いですが、最新の領事館情報を必ず確認してください。

※アポスティーユ（Apostille）の取得方法について：
日本国籍の方が日本の学校の証明書を提出する場合、外務省（東京本省または大阪分室）でアポスティーユ申請を行う必要があります。窓口での直接申請のほか、郵送での申請も可能です。手数料は無料ですが、返送までに約1〜2週間かかるため、ビザ申請のスケジュールを見据えて早めに入手しておく必要があります。

※英文残高証明書の注意点：
利用する日本の銀行によっては、英文での残高証明書の発行に数日から1週間程度かかる場合があります。また、残高証明書の発行日（基準日）から30日以上経過したものは領事館で受理されないため、あまり早く発行しすぎないよう注意が必要です。

## 4. ステップ3：ビザ申請の手順とタイムライン

必要書類がすべて揃ったら、住民票の住所を管轄する駐日大韓民国大使館、または領事館（総領事館）へ向かいます。東京などの一部地域では、ビザ申請業務を専門に扱う「KVAC（韓国ビザ申請センター）」での申請手続きが義務付けられている場合がありますので、事前に申請窓口を必ず確認しましょう。

以下は、留学準備からビザ取得、そして渡韓後の現地手続きまでの一般的なタイムラインと費用の目安です。

| プロセスの段階 | 実施時期（目安） | 費用の目安（日本円） |
| :--- | :--- | :--- |
| **大学・語学堂への出願** | 入学の2〜3ヶ月前 | 約5,000円〜15,000円（選考料） |
| **学費支払い＆入学許可書受領** | 入学の1.5〜2ヶ月前 | 各学校の規定による（授業料等） |
| **大使館・領事館でのビザ申請** | 出発の1〜1.5ヶ月前 | 日本国籍者は査証手数料無料（KVAC利用時は事務手数料約3,000〜4,000円） |
| **ビザ発給（審査期間）** | 申請後、約5〜10営業日 | なし |
| **外国人登録証（在留カード）の申請** | 韓国入国後90日以内 | 申請手数料 30,000ウォン（約3,300円） |

### スムーズな申請のためのアドバイス：
* **スケジュールに余裕を持って申請する:** 特に留学シーズン前（1月〜2月、7月〜8月）はビザの申請が殺到し、審査に通常より時間がかかることがあります。出発予定日の少なくとも6〜8週間前には書類を準備し始めることを強くお勧めします。
* **管轄領事館のWebサイトを毎週末チェックする:** 必要書類や申請予約のシステムは、予告なく変更されることが多々あります。
* **提出書類のコピー（またはスキャン）を取っておく:** 提出した書類は原則として返却されません。現地到着後、大学での最終手続きや外国人登録の申請時に、再度同じ情報の提示を求められる場合があるため、必ずデジタルデータ等で手元に残しておきましょう。

## 5. ステップ4：韓国入国後の必須手続き

おめでとうございます！無事にビザが発給され、韓国に到着したら、いよいよ楽しいキャンパスライフの始まりです。しかし、韓国で合法的に中長期滞在し、不便なく生活するためには、現地到着後に以下の手続きを迅速に行う必要があります。

* **外国人登録証（外国人登録、現在は在留カードと呼ばれる）の申請:** 
  韓国に90日を超えて滞在するすべての外国人は、入国後90日以内に、滞在先を管轄する出入国・外国人事務所で「外国人登録」を行い、外国人登録証を発行してもらう必要があります。
  登録には、韓国法務部のポータルサイト「HiKorea（ハイコリア：hikorea.go.kr）」での事前訪問予約が必須です。新学期の開始時期（3月、9月）は予約枠が数週間先まで埋まってしまうため、入国したらすぐに予約を確保しましょう。
* **国民健康保険（NHI）への自動加入:** 
  外国人留学生は、韓国入国後（外国人登録完了後）、韓国の国民健康保険に自動的に加入することが義務付けられています。毎月の保険料の請求書が登録された住所に届きます。この保険に加入することで、現地の病院で韓国人と同じ自己負担額で診療や治療、薬の処方を受けることができます。
* **アルバイト（時間外就業活動許可）のルールを遵守する:** 
  韓国では、学生ビザを持っているからといって、到着してすぐに自由に働けるわけではありません。不法就労とみなされないよう、以下の条件と手順を守る必要があります。
  * **D-2ビザ（正規学生）:** 入国後、最初の1学期（約6ヶ月）が終了した後に申請可能。学校の留学生担当部署の推薦と、出入国管理局からの「時間外就業活動許可」を得る必要があります。
  * **D-4ビザ（語学研修生）:** 入国後6ヶ月が経過し、前学期の出席率が90%以上である場合に申請可能。
  * **労働時間制限:** 週あたりの勤務可能時間は、保有するビザ、TOPIKの級、学校の成績等によって制限されます（例：通常週20時間までなど）。

※団体申請（代理申請）の活用：
多くの大学や語学堂では、学期の初めに留学生を対象とした外国人登録証の「団体申請（代理申請）」を学内で受け付けています。個人で出入国管理局を予約して訪問する手間が省けるため、学校から案内があった場合は積極的に活用することをお勧めします。ただし、団体申請の場合は個人で申請するよりも登録証の受け取りまでに時間がかかる（約1.5ヶ月〜2ヶ月）傾向があります。

## 6. 韓国留学ビザに関するよくある質問（FAQ）

韓国留学ビザの申請や維持に関して、多くの留学生が抱く疑問をまとめました。

### Q1. 語学留学（D-4）から大学進学（D-2）へビザを切り替えることはできますか？
はい、可能です。語学堂での研修中に、大学や大学院に合格した場合、韓国内の出入国管理局でD-4からD-2への資格変更申請を行うことができます。日本に一度帰国してビザを取り直す必要はありませんが、新しい大学の入学許可書や、十分な資金があることを証明する残高証明書などの書類を再度提出する必要があります。

### Q2. 銀行の残高証明書は親名義のものでも大丈夫ですか？
はい、本人以外の名義（通常は親など扶養家族）の残高証明書を使用することも可能です。ただしその場合、申請者と名義人の関係を証明する書類（日本の戸籍謄本など）の英訳、および家族関係証明の提出が追加で求められます。手続きをスムーズにするためには、可能な限り留学する本人名義の口座に資金を一時的に集めて残高証明書を発行することをお勧めします。

### Q3. 語学堂の出席率が下がるとビザはどうなりますか？
D-4ビザを維持する上で、最も重要なのが出席率です。出席率が90%を下回ると、次学期のビザ延長申請時に追加の説明書や事由書の提出を求められたり、延長が拒否されたりする可能性があります。また、出席率が70%〜80%未満になると、学校側から強制退学処分が下されることがあり、退学となった時点でビザの効力は失われ、速やかに出国しなければならなくなります。毎日の授業には必ず出席するよう心がけましょう。

### Q4. 学生ビザで入国後、別の国へ一時帰国や旅行をすることは可能ですか？
外国人登録証（ARC）の発給が完了した後は、事前の再入国許可を得ることなく、自由に韓国国外への旅行や一時帰国が可能です。ただし、外国人登録証が手元に届く前に韓国を出国してしまうと、せっかく取得した学生ビザ（シングルビザの場合）が無効になってしまいます。登録証の発行には1ヶ月程度かかるため、入国後最初の数ヶ月は、国外への移動スケジュールを入れないように注意してください。

### Q5. ビザ申請に「マルチビザ（数次査証）」は選べますか？
現在、韓国の学生ビザ（D-2、D-4）は、原則として「シングルビザ（一次査証）」として発給されます。しかし、現地に到着して「外国人登録」を完了すれば、その登録証の有効期間内であれば何度でも韓国を出入国できるようになるため、実質的にマルチプルビザと同等の役割を果たします。そのため、最初の申請時はシングルビザの発給で問題ありません。
