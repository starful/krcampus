# TOPIK（韓国語能力試験）完全攻略ガイド：日程・レベル・効率的な勉強法

韓国への進学や長期留学を計画していますか？ソウル大学、延世大学、高麗大学といった韓国の名門大学に入学するためにはもちろん、現地での生活をより豊かにするためにも、韓国語の習得は避けて通れません。その語学力を客観的に証明する最も権威ある世界基準の試験が、大韓民国政府（教育省）が認定・実施する**TOPIK（韓国語能力試験：Test of Proficiency in Korean）**です。

外国人留学生にとって、TOPIKスコアの取得は実質的な必須要件となっています。大半の韓国の大学では、学部課程（学士）の入学基準としてTOPIK 3級または4級以上、大学院課程（修士・博士）では4級以上を定めています。また、TOPIKの高スコアを保有していると、「大韓民国政府奨学金（GKS）」の選考で極めて有利になり、学費免除などの恩恵を受けやすくなります。さらに、D-2留学ビザの発給手続きがスムーズになるだけでなく、将来的に韓国で就職する際のビザ（D-10やF-2-7など）への移行時にも大幅な加点対象となります。

本ガイドでは、TOPIKの試験制度、級別の難易度、日本と韓国での受験日程、そして目標レベルを最短で突破するための実践的な学習戦略を徹底的に解説します。

## 1. TOPIKのレベルと評価基準：TOPIK I と TOPIK II の違い

TOPIKは、初級レベルを対象とした「TOPIK I」と、中・上級レベルを対象とした「TOPIK II」の2つの試験区分に分かれています。試験結果は、獲得した総合得点に応じて、1級（初心者）から最高峰の6級（上級者）までの判定が下されます。

まずは、各レベルが求める具体的な語学力と、留学や各種申請における重要性を確認しましょう。

| 試験区分 | 等級（級） | 合格基準点 | 求められる語学力・コミュニケーション能力の目安 | 留学やビザにおける重要性と位置づけ |
| :--- | :--- | :--- | :--- | :--- |
| **TOPIK I** | 1級 | 80点以上 | 自己紹介や買い物、飲食店の注文など、日常生活に必要な基本的な会話ができる。 | 交換留学の最低要件、または初期の学習指標。 |
| **TOPIK I** | 2級 | 140点以上 | 電話や依頼など、公共施設（郵便局、銀行など）を簡単なレベルで利用できる。 | 一部の専門学校や語学堂の入学要件。 |
| **TOPIK II** | 3級 | 120点以上 | 日常生活に不自由はなく、社会関係を維持するための基礎的な意思疎通ができる。 | **韓国の4年制大学（学部）の標準的な最低入学要件**。 |
| **TOPIK II** | 4級 | 150点以上 | 一般的なニュースや時事問題をある程度理解し、社会的なテーマで自分の意見を述べられる。 | **多くの大学で卒業条件とされる基準**。現地就職の基礎レベル。 |
| **TOPIK II** | 5級 | 190点以上 | 専門分野における研究や業務の遂行に必要な語学力を持ち、経済や社会のテーマを理解できる。 | 名門校への進学や、GKSなどの各種奨学金選考で非常に有利。 |
| **TOPIK II** | 6級 | 230点以上 | 専門分野の研究や実務をスムーズに遂行でき、ネイティブに近い表現や理解ができる。 | 大学院での高度な研究、韓国企業でのエグゼクティブ採用レベル。 |

TOPIK Iは基礎力確認として有益ですが、本格的に正規留学（学部・大学院への編入学や新入学）を果たすためには、最初から「TOPIK II」に照準を合わせ、最低でも3級、できれば4級以上の取得を目指す必要があります。

## 2. 試験構成、合格点、および受験にかかる費用

TOPIK IとTOPIK IIでは、試験時間、科目、問題形式に明確な違いがあります。TOPIK Iが全て選択式の客観式問題であるのに対し、TOPIK IIには受験生を大きくふるいにかける「書き取り（쓰기）」の記述式セクションが含まれています。

| 試験詳細 | TOPIK I（初級） | TOPIK II（中・上級） |
| :--- | :--- | :--- |
| **試験科目** | 聞き取り（40分）、読解（60分） | 【1時限目】聞き取り（60分）、書き取り（50分）<br>【2時限目】読解（70分） |
| **問題数と形式** | 全70問（すべて4択マークシート式） | 全104問（マークシート式100問 ＋ 記述式4問） |
| **満点** | 200点満点 | 300点満点 |
| **試験時間** | 100分（休憩なし） | 180分（2時限構成、合間に休憩あり） |
| **日本国内受験料** | 6,600円（税込） | 8,800円（税込） |
| **韓国現地受験料** | 40,000 KRW（約4,500円） | 55,000 KRW（約6,200円） |

*※注：日本国内での受験料は、主催する韓国教育財団の規定に基づきます。韓国現地で受験する場合は、ソウルや各地方の大学会場を選択でき安価ですが、手続きがすべて韓国語で行われるため迅速な申請が必要です。*

**【TOPIK II 書き取り問題（쓰기）の構成】**
TOPIK IIの書き取りは、合計100点の配点があり、高得点（5級・6級）獲得の命運を握ります。
*   **問51・52（短文空欄補充：各10点）**：電子メールや日常の案内文などの短い空欄に当てはまる適切な1文（約10〜20文字）を記述します。前後の文脈に合わせた自然な尊敬語や、文末の統一性（「-ㅂ/습니다」や「-ㄴ/는다」など）が問われます。
*   **問53（グラフ説明：30点）**：提示された図表や調査データについて、200〜300字で客観的に説明します。主観的な意見を述べず、データの変化傾向や原因、今後の展望を定型表現に当てはめて書くスキルが必要です。
*   **問54（長文論述：50点）**：与えられた社会的なトピックについて、自身の主張を根拠とともに600〜700字程度で論理的に執筆します。高度な語彙力と論理的な文章構造が厳密に評価されます。

## 3. TOPIKの年間試験日程と出願スケジュール

TOPIKを主催する韓国国立国際教育院（NIIED）は、世界各地で試験を実施しています。実施回数は受験地によって異なり、計画的な出願が必要不可欠です。

*   **日本と韓国での試験回数の違い：**
    韓国国内では**年6回**（通常1月、4月、5月、7月、10月、11月）実施されていますが、日本国内では**年3回**（通常4月、7月、10月）の実施にとどまります。志望する大学の願書提出締め切りから逆算して、受験スケジュールを早めに決めておきましょう。
*   **熾烈な出願期間：**
    申し込み期間は通常、試験実施日の約2〜3ヶ月前に設けられます。日本国内での受験申込は人気都市の会場が早期に満員になることがあります。また、韓国国内で受験する場合は、申し込み開始直後にアクセスが殺到して数分で枠が埋まる「チケッティング」状態となるため、事前にアカウント登録や個人情報を不備なく揃えておくことが求められます。
*   **成績発表と有効期限：**
    試験結果は通常、試験日から約4〜5週間後にオンラインで公開されます。成績証明書の有効期限は、**成績発表日からちょうど2年間**です。大学の出願日に1日でも期限が切れていると受理されないため、必ず確認してください。
*   **IBT（インターネット基盤テスト）の拡大：**
    近年、パソコンを使用して解答する「IBT TOPIK」の導入が韓国国内で進んでいます。従来の紙のテスト（PBT）に比べて成績発表までの期間が短縮されるというメリットがあり、今後日本を含む海外地域へのさらなる普及も期待されています。

## 4. TOPIK合格を勝ち取るための効果的な勉強法

TOPIKで最短で目標スコアを達成するには、ただ闇雲に語学学習をするだけでなく、試験に特化した攻略テクニックと日本人の強みを活かしたアプローチを組み合わせることが重要です。

*   **1. 過去問（기출문제）を制限時間通りに解く**
    最も王道かつ効果的な学習法は、公式から過去に公開された問題や市販の模擬試験を解くことです。実際の試験時間を正確に計りながら少なくとも過去5回分以上を解き、どのセクションで時間が不足するかを体感的に把握しましょう。自身の苦手な解答パターンを事前に分析・克服しておくことがスコアアップに直結します。
*   **2. 書き取り「問53」の万能テンプレートを暗記する**
    TOPIK IIの書き取りの中で、最も早く点数を伸ばせるのが問53（30点）です。この問題は、グラフの変化を客観的に記述する文言（例：「〜을/를 대상으로 조사한 결과에 따르면（〜を対象に調査した結果によると）」「〜에 비해 크게 증가하였다（〜に比べて大幅に増加した）」など）を丸暗記しておくことで、どのような問題が出題されても瞬時に完璧な文章を構成することができます。
*   **3. 漢字語（Sino-Korean）の関連性をフル活用する**
    日本語を母国語とする学習者にとって最大の武器は、韓国語の語彙の約60%以上を占める**「漢字語」**です。「学校（학교：ハッキョ）」「生活（생활：センファル）」のように、日本語の漢字の読みと韓国語のハングルの発音には明確な法則性があります。この漢字音の規則的な対応ルールを意識して勉強することで、読解セクションで見慣れない高度な単語が出てきた場合でも、文脈や漢字の意味から解答を推測することが十分に可能になります。
*   **4. 読解での時間配分の最適化とスキミング技術**
    TOPIK IIの読解は全50問あり、後半に進むにつれて学術的で抽象的な難易度の高い文章が並びます。全問を等しく精読していては時間が足りなくなるため、まずは比較的解きやすい問1〜25あたりまでを正確にノーミスで処理しましょう。その上で、設問の選択肢を先に読み、本文から必要な箇所を素早く探し出す「スキミング」の技術を磨く必要があります。
*   **5. 隙間時間に使える無料のオンラインリソースやアプリ**
    スマホアプリ（TOPIK用の単語・リスニングアプリなど）や、公式ウェブサイトの模擬試験機能を積極的に活用しましょう。通学時間や移動時の「毎日15〜30分」の継続的な聞き流しや単語チェックが、試験直前の詰め込みよりも強固な英語力（語学力）となって結実します。

## 5. 大学出願プロセスと留学ビザ申請への活用

取得したTOPIKスコアは、韓国への渡航スケジュールと緊密に連携させる必要があります。出願から渡韓までの全体的なプロセスを把握しておきましょう。

*   **韓国の大学への出願時期：**
    韓国の大学は一般的に3月入学（春学期）と9月入学（秋学期）があります。春学期の出願期間は「前年の9月〜10月頃」、秋学期の出願期間は「同年の3月〜4月頃」です。このオンライン申請のタイミングで、有効なTOPIK成績証明書を提出しなければなりません。タイミングを逃すと、丸半年から1年間入学が遅れることになるため十分な注意が必要です。
*   **D-2留学ビザの申請手続き：**
    大学から正式な「標準入学許可書（COA）」が発行された後、在日韓国大使館や各地域の領事館でD-2ビザを申請します。この際、公式なTOPIKのスコアカードを提示することで、韓国での就学能力や適性を公的に示すことができ、追加書類の要求や審査の遅れを避けることができます。
*   **語学堂（D-4ビザ）経由でのステップアップ：**
    現時点で目標とするTOPIKの等級に達していない場合でも、韓国への進路を諦める必要はありません。まずは大学付属の「語学堂」にD-4（一般研修）ビザで入学し、現地で毎日集中して韓国語を学び、TOPIKスコアを獲得した上で、大学（本科）のD-2ビザへ変更（現地での進学申請）をするルートが非常によく選ばれています。

## 6. TOPIK受験に関するよくある質問（FAQ）

### Q1. 試験当日に持参すべきものは何ですか？
**A1.** 必ず持参する必要があるのは、**①印刷した受験票、②有効な身分証明書（パスポート、マイナンバーカード、運転免許証など。学生証は認められない場合があります）、③修正テープ**です。筆記用具（専用の両頭ペン）は当日会場で全員に配布されるため、自身で筆記用具を準備する必要はありません。

### Q2. 試験中に自身のシャープペンシルや消しゴムは使用できますか？
**A2.** 解答用紙（マークシートや書き取り用紙）への記述は、会場で配布される専用の黒色サインペンのみを使用する決まりとなっています。下書き用の用紙や問題用紙へのメモにシャーペンを使うことは可能ですが、解答用紙の修正には各自が持参した**修正テープ**（修正液は使用不可）を使用しなければなりません。

### Q3. ハングル検定とTOPIKはどちらを優先して受けるべきですか？
**A3.** 日本国内の語学指標や一部の就職活動で評価されたい場合はハングル検定も意味がありますが、**韓国留学、現地の大学進学、奨学金の獲得、将来的な韓国企業での勤務や就職ビザの取得を少しでも目指すのであれば、最初から世界共通基準であるTOPIKを受験することを強くおすすめします。**

### Q4. 目標の級に届かなかった場合、成績はどうなりますか？
**A4.** TOPIK IIは3級〜6級の総合判定システムです。例えば合計120点未満だった場合は判定なし（不合格）となりますが、120点以上で3級、150点以上で4級のように、獲得した総合点数に応じて自動的にいずれかの級が認定されます。点数が足りずに完全に無効とされることはありませんので、実力測定のために積極的にチャレンジしましょう。

### Q5. 合格した後の成績証明書はどのように受け取りますか？
**A5.** 現在、紙ベースでの成績証明書の郵送は原則として行われていません。成績発表日以降、TOPIKの公式ホームページ（韓国のポータル）にログインし、ご自身でPDFファイルをダウンロードして印刷（無料）する仕組みとなっています。この印刷された証明書には公式な認証コードやQRコードが記載されており、大学や出入国管理事務所に正式な語学証明書類として提出できます。
