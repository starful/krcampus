# 慶熙大学校 国際教育院（IIE）：韓国語留学の完全ガイド

慶熙（キョンヒ）大学校は、1949年に設立された韓国屈指の名門私立大学です。その付属教育機関である「国際教育院（Institute of International Education: IIE）」は、1993年の設立以来、世界各国の留学生に向けて最高水準の韓国語教育と韓国文化体験プログラムを提供し続けています。年間2,000人以上の多様な国籍を持つ留学生が在籍するこの語学堂は、学問的な質の高さだけでなく、美しいゴシック様式のキャンパスでも広く知られています。

本ガイドでは、慶熙大学校国際教育院（以下、慶熙大語学堂）への留学を検討している方に向けて、カリキュラムの特徴、必要な学費、入学手続きからD-4ビザ申請のステップ、大学寄宿舎（学生寮）、そしてソウルでの生活環境まで、有益な情報を徹底的に解説します。

## 1. 学校概要
慶熙大学校国際教育院（IIE）は、ソウル特別市東大門区（トンデムング）の便利なロケーション（ソウル特別市東大門区慶熙大路26）に位置しています。回基（フェギ）駅から近く、治安が非常に良いため、学生が安心して勉学に専念できる環境です。

国際教育院が誇る主な魅力と特徴は以下の通りです。

* **歴史ある美しい名門キャンパス：** キャンパス内には、壮麗な石造りのゴシック建築「平和の殿堂」や本館などがあり、「韓国で最も美しいキャンパス」の一つに選ばれています。春は桜の名所（通称：慶熙ランド）としても非常に有名です。
* **D-4（一般研修）ビザ発給の完全サポート：** 2学期（約半年）以上の長期語学留学を希望する学生に対し、韓国政府公認のビザ申請用書類（標準入学許可書など）を迅速に発行します。
* **慶熙大生との活発なトウミ（バディ）制度：** 留学生一人ひとりに慶熙大学校の現役学生をマッチングする「トウミ制度」があり、日常会話の練習や韓国での生活サポートを受けられます。
* **TOPIK対策と充実した学習支援：** 公式TOPIK（韓国語能力試験）の試験会場に直接指定されているため、慣れ親しんだ環境で受験が可能です。試験対策プログラムも充実しています。
* **快適で高品質な大学寄宿舎の提供：** キャンパス内外に安全かつ快適な学生寮（Woo-Jung Wonなど）を完備。Wi-Fiやセキュリティも万全です。
* **韓国の伝統文化を体験できる多様なアクティビティ：** 言語学習だけでなく、テコンドー体験、K-POPダンス、伝統工芸、韓国料理教室、歴史遺跡へのフィールドトリップなどを通して、韓国への理解を深めることができます。

## 2. 課程・プログラムおよび日程
慶熙大学校国際教育院では、受講生の学習目的や期間に合わせて、多様なプログラムを提供しています。中心となる「正規課程」は、初級（1級）から上級（6級）までのレベルに分かれており、1学期10週間（計200時間）のカリキュラムが組まれています。

| プログラム名 | 入学時期 | 研修期間 | 授業時間 |
| :--- | :--- | :--- | :--- |
| **正規課程 (정규과정)** | 3月、6月、9月、12月 | 10週間 | 週20時間 (月〜金、1日4時間) |
| **速成課程 (속성과정)** | 3月、9月 | 10週間 | 短期間で集中して実力を高めるカリキュラム |
| **短期課程 (단기과정)** | 1月、8月 | 3週間 | 集中韓国語レッスン ＋ 文化体験 |
| **TOPIK対策クラス** | 随時 | 短期 | 試験対策・模擬試験・作文添削 |

### 各プログラムの特徴
* **正規課程（10週間プログラム）：** 読む・書く・聞く・話すの4技能をバランスよく鍛えるメインコース。毎日午前（9:00〜13:00）または午後（13:30〜17:30）の4時間、週5日行われ、各クラス15名前後の少人数制で、多国籍なクラスメイトとコミュニケーションを深めます。
* **速成課程：** 短期間で集中して実用的な語学力や口頭表現力を向上させたい学習者を対象としたインテンシブなカリキュラムです。
* **短期課程（3週間プログラム）：** 主に春休みや夏休みを利用して留学したい方向けに、1月と8月に開講。週20時間の韓国語レッスンに加え、さまざまな韓国文化体験が用意されています。
* **TOPIK対策クラス：** 試験対策に特化しており、読解・リスニング対策や、多くの学習者がつまずきやすい「作文問題」に対する個別添削指導を行います。

## 3. 学費および諸費用
学費は開講前の定められた期日までに、一括して大学指定の銀行口座に納付する必要があります。

| 費用項目 | 金額 (KRW) | 備考 |
| :--- | :--- | :--- |
| **選考料（入学申請料）** | 60,000 | 初回のみ、返金不可 |
| **正規課程 学費** | 1,750,000 | 1学期（10週間）分 |
| **速成課程 学費** | 1,450,000 | 1学期（10週間）分 |
| **短期課程 学費** | 980,000 | 1コース（3週間）分 |
| **TOPIK対策クラス学費** | 450,000 | 1セッション分 |
| **教材費（目安）** | 約 75,000 | 現地到着後に購入（レベルにより異なる） |

※ 上記の学費には、授業料のほか、主催される文化体験アクティビティの一部費用が含まれています。
※ 納付された選考料は、入学を辞退された場合でも一切返金されません。学費や諸費用は、大学側の事情によって変更される場合がありますので、必ず最新の募集要項を確認してください。また、海外送金の際の手数料は申請者の負担となります。

## 4. 入学手続きとD-4（一般研修）ビザ申請プロセス
韓国で2学期（約半年）以上語学留学をする場合、一般的に「D-4（一般研修）ビザ」を取得する必要があります。出願からビザ取得までのプロセスは以下の通りです。

1. **出願書類の提出：** 慶熙大学校国際教育院のオンラインシステムから申し込みを行い、以下の書類を提出します。
   * 入学願書（オンライン作成）
   * パスポートのコピー、証明写真（背景白）
   * 最終学歴の証明書（アポスティーユ認証、または大韓民国大使館の領事確認を受けたもの）
   * 預金残高証明書（本人または親名義で、10,000米ドル以上の残高証明。英語発行が推奨）
2. **書類審査：** 提出された書類を大学側で審査します（通常2〜3週間程度）。不備がある場合は再提出を求められます。
3. **学費の支払い：** 書類審査を通過後、送られてくる請求書に従って指定口座に学費を全額納付します。
4. **標準入学許可書（CoA）の受領：** 大学側での着金確認後、ビザ申請に必須の「標準入学許可書」が発行されます。
5. **D-4ビザの申請と発給：** 標準入学許可書、パスポート、ビザ申請書などを用意し、日本国内の大韓民国大使館・総領事館（またはビザ申請センター）でD-4ビザを申請します。発給には通常2〜4週間かかるため、余裕をもった計画が必要です。

## 5. TOPIK対策および大学・大学院進学パスウェイ
慶熙大学校国際教育院での学習は、将来韓国の大学や大学院への進学を志す留学生にとって有利な足がかりとなります。

### TOPIK（韓国語能力試験）への対応
国際教育院の正規課程では、時事ニュースの分析や学術的な討論、記述対策（作文添削）が通常のカリキュラムに組み込まれています。さらに、公式試験会場にもなっているため、普段勉強している慣れ親しんだ環境で本番の試験を受けられるという強みがあります。

### 慶熙大学校への進学優待（パスウェイ）
正規課程において、所定の要件を満たした修了生には、以下のような進学における特別な特典があります。
* **言語要件の免除：** 慶熙大学校の多くの学部や専攻科では、公式のTOPIKスコアに代わり、国際教育院の「正規課程3級（一部学科）または4級以上を修了」していることで語学要件を満たしたと認め、進学申請を受け付けています。
* **成績優秀者向けの奨学金：** 国際教育院在籍中の成績および出席率が特に優秀な学生が進学する場合、入学金や最初の学期の授業料が一部（または全額）免除される優秀者向け奨学金が適用されるチャンスがあります。
* **専門的なカウンセリングサポート：** 留学生専用の進学相談窓口や進学説明会が学内で定期的に開催され、願書の記入方法や面接対策などの指導を受けられます。

## 6. 寄宿舎（留学生寮）およびソウルでの生活環境
初めての海外生活でも安心できるよう、慶熙大学校では外国人留学生向けに快適で安全な生活環境を提供しています。

### Woo-Jung Won（友好館）などの大学寄宿舎
キャンパス至近の主要な寮である「Woo-Jung Won（ウジョンウォン）」は、高い人気を誇ります。
* **居室の設備：** 主に2人部屋（ダブルルーム）が提供され、室内には個別のベッド、机、椅子、クローゼット、個別エアコン、専用バスルーム（シャワー・トイレ）が完備されています。
* **共用スペース：** 建物内に自炊用の共用キッチン、ランドリールーム、スタディルーム、フィットネスジム、さらには24時間営業のコンビニエンスストアやカフェが併設されています。
* **費用：** 1学期（約4ヶ月）あたりの寮費は約1,200,000〜1,800,000 KRW程度です。保証金などの初期費用が安く抑えられるため、非常に安全かつ経済的な選択肢です。

### 周辺の生活環境（回基・東大門エリア）
大学の周辺一帯は「回基（フェギ）」と呼ばれる賑やかな大学街（学生街）です。
* **交通の利便性：** 地下鉄1号線・京義中央線が通る「回基駅」から徒歩約10〜15分、あるいは駅前のマウルバスを利用してスムーズにアクセスできます。東大門や明洞などの主要エリアへも地下鉄で20〜30分程度です。
* **物価と生活：** 学生向けのリーズナブルで美味しい飲食店や、お洒落なカフェが密集しています。スーパー、100円ショップのダイソー、コスメショップ（オリーブヤング）に加え、近くに病院もあるため、日常生活で不便を感じることはありません。

## 7. よくある質問（FAQ）

### Q1. D-4（一般研修）ビザでアルバイトをすることはできますか？
はい、可能です。ただし、入国後すぐに働けるわけではありません。語学堂に「6ヶ月以上」在籍し、前学期の平均出席率が「90%以上」かつ学業成績が良好であることが条件となります。その上で、学校および管轄の出入国管理事務所から正式に「資格外活動許可」を受ける必要があります。無許可での就労は不法就労として厳罰に処されますのでご注意ください。

### Q2. 出願時に支払った選考料（60,000 KRW）は返金されますか？
いいえ。選考料（入学登録費）の60,000 KRWは、書類審査やシステムへの登録など事務処理の費用となるため、合否結果や自己都合によるキャンセルの有無を問わず、返金されません。

### Q3. 韓国語をまったく話せない初心者ですが、授業についていけますか？ またクラス分けはどのように行われますか？
心配はいりません。入学手続きを終えたすべての学生は、最初の授業が始まる前に必ず「プレイスメントテスト（レベル分けテスト：筆記および簡単な口頭面接）」を受けます。これによって、アルファベット（ハングル）の読み書きから始める「1級（初級）」から、流暢な日常会話を目指す「6級（上級）」まで、各自の実力に最適なクラスに配属されます。初級クラスは初めての方でも分かりやすく丁寧に授業が進められます。

### Q4. 留学中、現地の医療保険への加入は義務ですか？
はい、義務付けられています。韓国政府の法律に基づき、外国人登録証を発行したすべての長期留学生は、韓国の「国民健康保険（NHIS）」に自動的に加入することになります。これにより、韓国内の病院を診察・治療目的で受診する際、少額の自己負担額で医療サービスを受けられます。国民健康保険が適用されるまでの入国初期（約2〜3ヶ月）については、日本出国前に民間の海外旅行保険や海外留学生保険に加入しておくことを強くお勧めします。

### Q5. 進級やビザ維持のための出席率の条件はありますか？
はい、あります。無事に修了して次の学期へ進級し、かつD-4ビザを更新するためには、該当学期の授業で「出席率80%以上」および「総合評価が60点（100点満点中）以上」を満たす必要があります。出席率が80%を割り込むと、いかなる場合も「未修了（留年）」扱いとなります。また、出席状況が著しく悪い場合は、学校側からビザの更新申請が拒否され、その時点で日本に帰国せざるを得なくなるため、自己管理が極めて重要です。
